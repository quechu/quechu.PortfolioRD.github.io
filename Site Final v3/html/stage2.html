Application en Windev

-Contexte

Imaginez, vous etes dirigeant d'une entreprise, vous avez beaucoups de responsabilités
et malheureusement peu de temps à consacrer aux différentes taches, notamment ce qui
est le plus important la gestion des opportunités de votre entreprise.
Découvrez une application en Visual Basic qui vous permet en un coup d'oeil de savoir comment fonctionne votre entreprise.

Cahier des charges

  1. Créer une application sous Windev permettant de lire un fichier txt.
  2. Retranscrire ces données dans un fichier au format excel.
  3. Trier ces données puis les retranscrire dans un fichier txt.
  4. Créer une application d'emailing basé sur ces fichier afin de faire des rappel.
  5. Mise en place de l'application sur le serveur local.
