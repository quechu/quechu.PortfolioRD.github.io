-Projet

Xamarins

-Contexte

App Movies :
# Il s’agit de proposer une application Android, où il suffira de taper un mots-clés qui décrivent
  le film pour que le moteur de recherche le titre et un descriptif en question.
# Cette application va s’appuyer sur une API « The Movie Database ».

App Musée des Célestins :
# Le Musée des Célestins souhaite une application Android qui permette de consulter l’ensemble
 de ses oeuvres, classées suivant des critères.
# Puis, après avoir choisi un peintre, obtenir l’ensemble de ses oeuvres (photographie, puis descriptif)
 exposées dans le musée, ainsi que sa biographie.
# Dans un premier temps il s’agira de développer la partie CLIENTE sous la forme des données en JSON et HTML.
