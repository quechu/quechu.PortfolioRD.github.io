Maintenance EVOLUTIVE des applications de saisie de questionnaires (web) et de CREATION DYNAMIQUE D’UNE INTERFACE (C#) à partir d’un document XML

-Projet

WinForms / C#

-Contexte

La société ALLIER TESTS, implantée dans Vichy, intervient dans la mise en oeuvre informatique
de QUESTIONNAIRES qu’elle propose en tant que TESTS à ses sociétés clientes.
A la demande de ses clients, elle saisit les QUESTIONNAIRES et les QUESTIONS associées
dans une base de données MySQL, en utilisant sa plate-forme Intranet (HTML, CSS et PHP).
ALLIER TESTS fait appel à la SIII « SIVY » afin d’assurer la MAINTENANCE EVOLUTIVE de l’application existante.

-Code

Cahier des charges

  1. Développement d’une fonctionnalité PHP d’extraction des données au format XML.

  2. Adaptation de l’application existante pour pouvoir traiter tous types de contrôles graphiques.

  3. Mise en œuvre d’une interface MDI, puis enregistrement de la réponse dans la base de données.
