Maintenance évolutive du logiciel de gestion des cartes grises

-Projet

PHP / SGBDR MySQL

-Contexte

Il s’agit de développer une fonctionnalité supplémentaire : une interface
de gestion des données relatives aux voitures que
les commerciaux utilisent lors de leurs déplacements.

-Code

Cahier des charges

  1. Création du schéma de base de données et d’un script PHP, partiellement développé, pour l’affichage des modèles de véhicules.

  2. Développement des scripts permettant
    1. Insertion de nouvelles données dans la table « Modèle ».
    2. Insertion simultanée des données d’une voiture et de sa carte grise.
    3. Retrouver tous les propriétaires d’un modèle donné.

  3. Conception et réalisation d’un formulaire PHP permettant de consulter les tables de la base.
